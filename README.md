# CompMathProj

Este repositório contém projetos de matemática computacional realizados em MATLAB.

No interior das pastas encontras os enunciados e as respostas aos exercícios pedidos na disciplina.

- `Projeto1/` e `Projeto2/`: exercícios e soluções em MATLAB.

Os ficheiros `.m` são os códigos MATLAB com as respostas e métodos implementados.

## Temas abordados

**Projeto1:**
- Resolução de equações não lineares usando métodos iterativos (exemplo: método de Newton para aproximação de ângulos em mecanismos, como na função "freud").

**Projeto2:**
- Cálculo de derivadas e análise de sinais de derivadas de funções.
- Integração numérica: métodos dos trapézios e de Simpson para aproximação de áreas.
- Ajuste de curvas (regressão linear) para determinar coeficientes de funções a partir de dados.
